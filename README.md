# Glass Box

Glass Box is a UX-first framework for handling AI uncertainty — built to orchestrate trust with spatial reasoning UI primitives and a strictly typed DAG state core.

## Monorepo layout

- `apps/docs`: Next.js documentation site
- `apps/playground`: Next.js local testing environment
- `packages/core`: Pure TypeScript DAG state logic
- `packages/react`: React UI component library (Tailwind + Framer Motion)
- `packages/theme`: Shared design tokens (colors, spacing) + Tailwind preset

## Getting started

```bash
npm install
npm run dev
```
